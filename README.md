# Installation
Clone this repository:  
> git clone https://github.com/2022-CVPR-8343/MLCN.git  
> cd ./MLCN/

Create a virtual envs:
> conda create -n [ENV] python=3.8  
> conda activate [ENV]

Install PyTorch and other dependencies:  
> pytorch == 1.8  
> torchvision == 0.9.0  
> opencv == 4.5.3  

Install deformable convolution:  
> sh make.sh
